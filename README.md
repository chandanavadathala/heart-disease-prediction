# ❤️ Heart Disease Prediction App

This is a Machine Learning web application that predicts the likelihood of heart disease using patient data.

## 🚀 Features

* Predicts heart disease using ML model
* Interactive UI using Streamlit
* Real-time prediction

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy

## 📂 Project Files

* app.py → Streamlit app
* logistic_heart.pkl → trained model
* scaler.pkl → data scaler
* columns.pkl → feature columns

## ▶️ Run Locally

pip install -r requirements.txt
python -m streamlit run app.py

Includes data analysis and model training:
